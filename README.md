# Luke Game

### Install:

- clang
- cmake
- libsdl2-dev
- libsdl2-image-dev
- libsdl2-ttf-dev
- libsdl2-mixer-dev
- libsdl2-gfx-dev

Instructions

- use wsl
- sudo apt get install     for those above

- go into build directory and type 'cmake ..' //compiles
- type 'make' //builds 
- type './luke_game'
